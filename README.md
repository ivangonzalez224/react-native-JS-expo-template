## 💻 Getting Started <a name="getting-started"></a>

In this project it is used React Native to crerate a template using JavaScript and Expo.

### Prerequisites

In order to run this project you need:

1. Git.
2. Visual Studio Code or another code editor.
3. Browser.

### Setup

To get a local copy, follow these steps.

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/ivangonzalez224/react-native-JS-expo-template.git
```

### Install

This project does not require installation. Just clone the repository to your local machine.
Even though, for general purposes, Install this project with:
```sh
  cd my-folder
  npm install
```

### Usage

To run the project, do the following:
- Run `npm start` to start the development server to open an iOS or Android simulator.
- Besides, download de the expo app to try in real time your development.
